Matter QR Code Tool

This tool lets you upload a Matter QR code, decode it, and recover the setup code.You can then generate clean SVG or STL files, ready for 3D printing or creating replacement labels.

Everything runs locally in your browser. No sensitive data is sent anywhere.

1. Scan Or Upload

Live Camera

Point your phone or laptop camera at the Matter QR code for fast scanning.

Photo Upload

Choose an existing photo or screenshot if the live scanner is not convenient.

If automatic decoding still does not work, you can paste the MT: payload directly into the field in step 2.

2. Review Payload

- Waiting For Input

The MT: value is the Matter setup payload stored in the QR code. It contains the commissioning data used to pair the device, so it should be treated as sensitive.

Extracted Pairing Code
-

Optional: This requests the product records via the extracted vendor and product IDs.

No sensitive information is sent. Only the extracted vendor and product IDs are used for the lookup, not the setup PIN or full MT: payload.

Warning: do not share the extracted QR contents or pairing code with anyone you do not trust.

Instructions for Bambu Lab / OrcaSlicer

  • Generate and download the QR file you want to use.
  • Right-click the part you want to modify.
  • Choose Add modifier, then Load, and select the QR SVG or STL.
  • Position the modifier where you want the QR.
  • Change the filament color so it shows up on the finished print.
  • If the final visible QR would end up reversed, enable mirroring before exporting.
