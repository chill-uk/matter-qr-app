Matter QR Code Tool

This tool lets you upload a Matter QR code, decode it, and recover the setup code.You can then generate clean SVG or STL files, ready for 3D printing or creating replacement labels.

Everything runs locally in your browser. No sensitive data is sent anywhere. The optional live CSA DCL lookup only requests the vendor and product records needed for the extracted IDs.

1. Upload Photo & Decode

Upload a photo of the device's Matter QR code to extract the setup payload.

If photo decoding does not find a QR code, you can scan the label with a third-party QR app, copy the MT: value, and paste it into the field in step 2.

2. Review Payload

- Waiting For Input

The MT: value is the Matter setup payload stored in the QR code. It contains the commissioning data used to pair the device, so it should be treated as sensitive.

Extracted Pairing Code
-

Optional: query the official CSA Distributed Compliance Ledger for live vendor and model records. This requests the matching vendor and product records for the extracted IDs, not the setup PIN or full QR payload.

Warning: do not share the extracted QR contents or pairing code with anyone you do not trust.

3. Export

Instructions for Bambu Lab / OrcaSlicer

  • Generate and download the QR code SVG.
  • Right-click the part you want to modify.
  • Choose Add modifier, then Load, and select the QR-code SVG.
  • Position the modifier where you want the QR.
  • Change the filament color so it shows up on the finished print.
  • If the final visible QR would end up reversed, enable the mirror option before exporting.
